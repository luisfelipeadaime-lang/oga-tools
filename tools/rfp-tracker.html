RFP Tracker
How to use ›
  1. Select an RFP from the list above.
  2. Review criteria verbatim in the Criteria tab — nothing is paraphrased.
  3. Add projects under evaluation in the Projects tab.
  4. Upload supply docs per project (PDD, deck, commercial terms).
  5. Click Run Evaluation — AI scores each criterion against your docs. Criterion text is always shown verbatim. Never a rewrite.
  6. Use Submission Prep tab to track what’s needed before submitting to Typeform.
